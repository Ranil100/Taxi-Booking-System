## Overview
The **Taxi Booking System** is a simple console-based Java application that allows users to book taxis, calculate fares, and manage ride details.  
It demonstrates the use of **Object-Oriented Programming (OOP)** concepts such as classes, inheritance, and encapsulation.

---

## Features
- Book a taxi by entering pickup and drop locations  
- Calculate fare based on distance  
- View booking details  
- Manage driver and ride records  

---

## Concepts Used
- Classes and Objects  
- Inheritance and Polymorphism  
- Encapsulation  
- Exception Handling  
- Java Collections (ArrayList, HashMap)

---

## Tech Stack
- **Language:** Java  
- **IDE:** IntelliJ IDEA / Eclipse / VS Code  

---

## How to Run
1. Clone or download the project.  
2. Open it in your preferred Java IDE.  
3. Run the `Main.java` file.  
4. Follow the console instructions to use the system.
